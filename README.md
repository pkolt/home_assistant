# Умный дом с Home Assistant

https://www.home-assistant.io/installation/linux/#survey_section

http://127.0.0.1
http://127.0.0.1:8123

## Запуск

```sh
docker compose up -d
```

## Остановить контейнеры

```sh
docker-compose stop
```


## Остановить и удалить контейнеры

```sh
docker compose down --rmi local -v
```

## Запустить снова

```sh
docker-compose start
```
