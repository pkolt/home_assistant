# Умный дом с Home Assistant

Подробнее про [установку Home Assistant](https://www.home-assistant.io/installation/linux/#survey_section).

http://127.0.0.1
http://127.0.0.1:8123

## Создание и запуск

```sh
docker compose up -d
```

## Остановить

```sh
docker compose stop
```


## Остановить и удалить

```sh
docker compose down --rmi local -v
```

## Запустить

```sh
docker compose start
```
